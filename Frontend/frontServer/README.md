# frontserver

> A Vue.js project

## Build Setup

``` bash
# install dependencies(애플리케이션 실행을 위해서 필요함)
npm install

# serve with hot reload at localhost:8080
npm run dev


```

## 코드위치

``` 
src / App.vue(전체화면)
    / FormTemplate.vue (환율정보 보여주는 결과화면)
    / main.js(App.vue 실행을 위한 밑바탕 js, lib import에 사용함)


```
